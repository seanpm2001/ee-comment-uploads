Installation for Comment Uploads 2.0.x on EE 1.x

These instructions are for installing Comment Uploads 2.0.x on ExpressionEngine 1.x only. If you're installing the latest version of Comment Uploads, please refer to the applicable installation instructions.

1. Copy Add-on Files

  • Within the /EE1/system/ directory, copy the bridge folder into your sites' ./system/ directory.
  • Within the /EE1/system/extensions/ directory, copy all folders and files into your sites' ./system/extensions/ directory.
  • Within the /EE1/system/language/english/ directory, copy the lang.comment_uploads.php file into your sites' ./system/language/english/ directory.

2. Installing the Add-On

  • Go to the Extensions area in the ExpressionEngine Control Panel, find Comment Uploads in the list, and click Enable.

3. Setup & Configuring

  • Enabling the Comment Uploads extension will automatically create a new File Upload Preference in the ExpressionEngine CP, and a new folder named comments in your ./images/uploads/ directory.
  • Make sure the permissions of this folder to writable, typically 777 on Unix servers (Note:EngineHosting clients can skip this step).
  • There is no CP-based configuration for the Comment Uploads extension, aside from the EE File Upload Preference - you can edit that as necessary. The rest of the functionality lies with the template tags.
